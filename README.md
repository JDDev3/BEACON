
# BEACON


The goal of BEACON is to provide users with access to list of curated clinical datasets in bladder cancer

## Installation

You can install the development version of BEACON from [GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("JDDev3/BEACON")
```
or with: 
```r
#install.packages("remotes")
remotes::install_github("JDDev3/BEACON")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(BEACON)
## basic example code
```
